# Favorite Things Party

A Netlify-ready party website for Stephanie and Jeff's annual Favorite Things party.

## Features

- Party invite and details
- Netlify Forms RSVP
- Gift exchange rules
- Interactive games hub
- Polaroid photo booth
- Shared gallery preview

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`

## Notes

The photo booth currently stores gallery items locally in the browser. The code is structured so a shared backend/storage service can be added next for cross-device gallery persistence.
